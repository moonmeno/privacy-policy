개인정보 처리방침

최종 업데이트: 2025년 1월 1일

서비스명: Social Comment Manager

연락처: support@yourdomain.com

1. 개요

Social Comment Manager는 YouTube 및 Instagram 댓글을 관리하는 웹 기반 서비스입니다.

2. 수집하는 정보

Google (YouTube) 연동 시

  • 기본 정보: 이름, 이메일 주소, 프로필 사진
  • YouTube 채널 정보: 채널 ID, 채널명, 구독자 수
  • YouTube 댓글: 댓글 내용, 작성자명, 작성일
  • OAuth 액세스 토큰

Meta (Instagram) 연동 시

  • Facebook 페이지 정보
  • Instagram 계정 정보
  • Instagram 댓글
  • 액세스 토큰

3. 정보 사용 목적

  • 사용자 인증 및 로그인
  • YouTube/Instagram 댓글 조회 및 관리
  • 댓글 자동 모더레이션
  • 서비스 개선

4. 정보 공유

본 서비스는 사용자의 개인정보를 제3자에게 판매하거나 마케팅 목적으로 공유하지 않습니다.

제3자 서비스

  • Google (YouTube): OAuth 인증 및 API 사용
  • Meta (Instagram): OAuth 인증 및 API 사용

5. 데이터 보안

  • HTTPS/TLS 암호화
  • 데이터베이스 암호화 (AES-256)
  • 액세스 토큰 암호화 저장

6. 사용자 권리

  • 접근 권리: 대시보드에서 본인 정보 확인
  • 수정 권리: 정보 수정 가능
  • 삭제 권리: 계정 삭제 요청 (30일 이내 처리)
  • 데이터 다운로드: 데이터 내보내기 가능

7. 보관 기간

  • 계정 정보: 계정 삭제 시까지
  • 댓글 데이터: 연동 해제 후 30일
  • 액세스 토큰: 만료 또는 해제 시 즉시 삭제

8. 연락처

개인정보 관련 문의:

  • 이메일: privacy@yourdomain.com
  • 응답 시간: 영업일 기준 48시간 이내

9. 정책 변경

본 개인정보 처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경 시 30일 전 이메일 공지합니다.
