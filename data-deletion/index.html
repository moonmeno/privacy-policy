데이터 삭제 정책

최종 업데이트: 2025년 1월 1일

1. 개요

Social Comment Manager는 사용자의 "잊혀질 권리"를 존중합니다.

2. 삭제 가능한 데이터

  • 사용자 계정 정보
  • Google (YouTube) 연동 정보
  • Meta (Instagram) 연동 정보
  • 댓글 데이터
  • OAuth 토큰
  • 모더레이션 규칙

3. 데이터 삭제 방법

방법 1: 대시보드에서 직접 삭제

  1. 로그인 후 대시보드 접속
  2. 설정 → 계정 관리
  3. "계정 삭제" 버튼 클릭
  4. 확인 후 "DELETE" 입력
  5. 최종 삭제 버튼 클릭

처리 기간: 즉시 비활성화, 7일 이내 완전 삭제

방법 2: 이메일 요청

이메일: anstkswjd@gmail.com

제목: 계정 삭제 요청

본문: 이름, 이메일, 요청 사유

처리 기간: 7 영업일 이내

방법 3: Google/Meta에서 직접 삭제

Google:

  1. Google 계정 설정
  2. "타사 앱" → "Social Comment Manager" 삭제

Meta:

  1. Facebook 설정
  2. "비즈니스 통합" → "Social Comment Manager" 삭제

4. 삭제 타임라인

  • 즉시 (0일): OAuth 토큰 삭제, 계정 비활성화
  • 7일 이내: 사용자 정보, 댓글 데이터 삭제
  • 30일 이내: 백업 데이터 완전 삭제

5. 삭제되지 않는 데이터

  • YouTube/Instagram 원본 댓글 (플랫폼에서 직접 삭제 필요)
  • 익명화된 통계 데이터
  • 법적 의무 보관 데이터
